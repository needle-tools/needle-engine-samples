# Screen Sharing

Add real-time screen sharing to a multiplayer 3D scene — your own Zoom, built in the browser with Needle Engine. Share your screen, a window, or a webcam with everyone in the room, audio included, streamed live over WebRTC.

The `ScreenCapture` component handles the stream and pairs with a `VideoPlayer` component that decides where the video appears in your scene — a wall, a floating panel, or any surface. Because it's networked, every connected user sees the shared feed live.

**Learn more**
- [ScreenCapture](https://engine.needle.tools/docs/api/ScreenCapture) — API reference
- [VideoPlayer](https://engine.needle.tools/docs/api/VideoPlayer) — render the stream in your scene
