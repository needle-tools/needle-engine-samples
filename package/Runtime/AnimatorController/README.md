# Animator Controller

An Animator Controller is a powerful state machine that drives your animations — deciding what plays, when, and how it blends from one animation (a **State**) to the next. It scales from a simple open/close toggle all the way up to full character behavior, and it runs on the web with no code required.

This sample opens and closes a treasure crate using a trigger called **Toggle**. Double-click the `Crate AnimatorController` asset to explore the states and transitions, and click any of them to tweak their settings.

Prefer to build controllers from code? The [`AnimatorControllerBuilder`](https://engine.needle.tools/docs/api/AnimatorControllerBuilder) gives you a fluent, type-safe way to create them programmatically.
