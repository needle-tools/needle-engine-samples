# Animator Controller

An Animator Controller is a state machine for animation — it decides what plays, when, and how it blends from one animation (a **State**) to the next. The same approach scales from a simple open/close toggle to full character animation, and it runs on the web with no code required.

This sample opens and closes a treasure crate with a trigger called **Toggle**. Double-click the `Crate AnimatorController` asset to explore the states and transitions, and click any of them to tweak their settings.

Prefer to build controllers from code? The [`AnimatorControllerBuilder`](https://engine.needle.tools/docs/api/AnimatorControllerBuilder) gives you a fluent, type-safe way to create them programmatically.
