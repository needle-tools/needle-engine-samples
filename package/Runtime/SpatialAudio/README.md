# Spatial Audio

Position sound in 3D so it gets louder as you move closer and shifts as you turn — believable spatial audio on the web, in the browser and in VR/AR. Each `AudioSource` sits at a point in the scene with natural distance falloff, so a room full of instruments sounds like it's really around you.

Needle Engine audio blends seamlessly from 2D to 3D with a single slider, so you can mix both in one scene — flat, everywhere sound like music or narration alongside positional 3D sound, dialed to exactly the balance you want.

This sample places playable instruments you can move around. It's built with Needle Engine Everywhere Actions — no code required — so it also works in iOS AR and on Android. Try it in a headset for the full effect.
