# Collaborative Sandbox

Build a real-time multiplayer sandbox on the web where people join from any browser, phone, or VR/AR headset and arrange a shared 3D scene together. It's a ready-made starting point for collaborative building tools, playful world editors, and multiplayer WebXR experiences.

Under the hood, drag-to-move and click-to-duplicate let users pick from predefined parts and place them freely, while networked transforms keep every object's position in sync across all connected clients. Ownership is handled automatically: whoever grabs an object takes control of it while dragging.

- Cross-platform drag and duplicate, from desktop to WebXR
- Live position sync so everyone sees changes instantly
- Runs in AR and VR out of the box
