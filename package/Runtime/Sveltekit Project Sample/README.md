# SvelteKit Sample

Build a full web app around an interactive 3D scene with Needle Engine and SvelteKit — multiple routes, server integration, or static site generation, with your 3D content embedded throughout.

The sample uses `adapter-static` to output a statically generated site you can deploy anywhere. Switch to `adapter-auto` when you need dynamic routes or a backend server.
