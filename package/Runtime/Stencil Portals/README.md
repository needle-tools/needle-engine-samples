# Portal Rendering

Build impossible, non-euclidean spaces on the web — rooms that hold whole worlds, windows into other dimensions, and Antichamber-style scenes that break the rules of real geometry. This portal rendering sample shows how to open a view into content that only exists when you look through it.

It works with stencil masking: a mask shape marks part of the screen, and objects tagged as that mask's "content" render only inside it. The sample ships one portal wired up but is pre-configured for three separate worlds, so you can layer more portals for richer effects.

- Runs in the browser on top of three.js, with Unity URP export
- Look through and orbit each portal in real time
- Extend from 1 to 3 independent masked worlds
