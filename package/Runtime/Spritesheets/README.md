# Spritesheets

Play 2D sprite sheet animations in the browser — animated characters, effects, or UI, running on desktop, mobile, and XR. Sprite animations you author in Unity export straight to the web, with no code required.

This sample plays a sprite animation with the **SpriteRenderer** component driven by an **Animator**. You create the sprite sheet and its **AnimationClips** in Unity; Needle exports them, and at runtime the Animator plays the frames like any other animation state.
