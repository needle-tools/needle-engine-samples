# React

Add an interactive 3D scene to a React app and pass events between your React UI and the scene. If you already build with React, this is how you bring real-time 3D and WebXR into your stack without leaving it.

This sample mounts Needle Engine inside a React component and exposes an API so the two sides can talk to each other: a click in your React interface can drive the 3D scene, and events from the scene can update your React state.
