# Bow & Arrow

A VR archery minigame for the web — grab the bow, draw the string, and shoot targets, with real physics-based arrows. It also runs on desktop and mobile, so the same game works everywhere from one project.

Draw power comes from the distance between your hands, arrows fly with `Rigidbody` physics and collide with spawned targets, and hits react with particle effects and sound. It's a complete, dissectable starting point: the sample scripts (`ArrowShooting`, `Arrow`, `BowArrowTarget`, `BowTargetSpawner`) handle shooting, projectile behavior, and target spawning — ready for you to add scoring or new target types.

**Learn more**
- [XRControllerFollow](https://engine.needle.tools/docs/api/XRControllerFollow) — attach the bow to a controller
- [Rigidbody](https://engine.needle.tools/docs/api/Rigidbody) — arrow physics
