# Everywhere Actions

Build interactive AR experiences that run **everywhere** — including native iOS Augmented Reality via QuickLook — without writing a line of code. Everywhere Actions are a set of Needle Engine components you add in Unity to make objects clickable, animated, and interactive.

The interactive USDZ is generated **at runtime**, right in the browser, rather than pre-baked at build time. That keeps things lightweight and dynamic: the scene can change — swap a product variant, update content — and the exported USDZ reflects it, so the same project works in iOS Quick Look, on Android, and on the web.

The scene bundles several focused examples:

- **USDZ Interactivity** — the basic actions shown in isolated use cases.
- **USDZ Characters** — relative-movement actions that make minigames playable everywhere, including iOS AR.
- **USDZ Animator** — a skinned, animated character that works across platforms.
- **USDZ Product** — a simple product configurator with high-quality materials, auto-converted to QuickLook / USDZ.
- **Musical Instrument** — animation and audio playback on click.
