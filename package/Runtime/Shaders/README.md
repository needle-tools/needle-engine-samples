# Shaders

Bring your own custom Unity shaders to the web. This sample shows how materials authored in Unity's Shader Graph run in real time in the browser, so you can ship animated, stylized, and effect-driven looks without rewriting them in GLSL.

Each material is built as a node graph in Unity, then exported to glTF and rendered live on the web. Explore the collection to see how different Shader Graph setups translate into interactive web visuals.

- Author shaders visually in Shader Graph, no low-level GPU code required
- Animated and time-based effects that keep running in the browser
- One export path from Unity materials to a live web page
