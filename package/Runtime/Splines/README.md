# Splines

A spline defines a smooth path through 3D space. Use one to animate an object along a predefined route — with an AnimationClip, or driven live by user input like page scroll.

A common use is a camera that follows a spline and reacts to scroll as visitors move down a website, revealing the scene section by section. That scroll-driven spline camera is exactly how [needle.tools](https://needle.tools) is built.

**Learn more**
- [SplineContainer](https://engine.needle.tools/docs/api/SplineContainer) — define the path
- [SplineWalker](https://engine.needle.tools/docs/api/SplineWalker) — move an object along a spline
- [ScrollFollow](https://engine.needle.tools/docs/api/ScrollFollow) — drive it from page scroll
