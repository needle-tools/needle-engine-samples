# Shooting Range

A browser-based FPS game: score the most points by landing as many shots as you can without missing. It runs on PC, mobile, and VR from a single project — the same first-person game, playable with keyboard and mouse, touch controls, or motion controllers.

It's a complete starting point you can dissect and build your own game from: a character controller, a gun and targets, scoring, and cross-platform input all wired together with Needle Engine.
