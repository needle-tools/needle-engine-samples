# Voxel Editor

Build blocks in your browser with this online voxel editor: click to add and remove cubes, then view your creation in Augmented Reality on iOS. It's a starting point for your own voxel maker, level builder, or any tap-to-edit 3D tool that runs on the web without an install.

The whole editor is built entirely with Everywhere Actions, a set of no-code Needle Engine components. Clicks are detected by a raycaster, and each voxel toggles its neighbors on or off, so the same interactions work identically in the browser and in iOS QuickLook AR.
