# Vue.js Integration

Add an interactive 3D scene to a Vue.js app with Needle Engine. This sample drops the `<needle-engine>` web component straight into a Vue component, so your 3D content and your Vue UI live in one project — HTML layers render in front of or behind the 3D via z-index, and Vue can drive the scene directly.

It also shows how a Unity-authored Animator and animation state machine can be controlled from the Vue frontend, and it's networked out of the box: share the room URL (or open a second window) and the current slide stays in sync for everyone on the page. The Vue project template is fetched from GitHub when you run the scene in Unity.
