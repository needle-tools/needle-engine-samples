# Musical Instrument

Tap the parts of a 3D instrument to play sounds and animations — an interactive musical instrument on the web, built entirely no-code with [Everywhere Actions](https://engine.needle.tools/docs/how-to-guides/everywhere-actions/). Because it exports as interactive USDZ, it also plays in AR Quick Look on iOS.

Each part uses `PlayAudioOnClick` and `PlayAnimationOnClick`, so a tap fires a sound and a matching animation at the same time — all configured in the Unity editor, no scripting. Swap in your own model and sounds to turn any object into a clickable, playable instrument.
