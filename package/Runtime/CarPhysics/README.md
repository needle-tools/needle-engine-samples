# Car Physics

Add drivable car physics to any web project. Drop in a 3D model and Needle Engine turns it into a controllable vehicle you can steer around a browser scene with keyboard, gamepad, or touch — no plugins, built on three.js.

The sample demonstrates raycast vehicle physics: each wheel casts a ray to the ground for suspension, grip, and steering, so cars stay stable at speed. Automatic wheel detection rigs most models instantly, or falls back to a default car setup.

- Works with any imported model via automatic car detection
- Keyboard, gamepad, and mobile touch controls out of the box
- Streamable in-car radio and drag-and-drop-to-drive

**Learn more**
- [Full example on GitHub](https://github.com/needle-engine/car-physics)
- [MeshCollider](https://engine.needle.tools/docs/api/MeshCollider) — collision from track and environment geometry
- [Camera](https://engine.needle.tools/docs/api/Camera) — follow the car through the scene
