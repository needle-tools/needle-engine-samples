# Product Flyover

Create cinematic product reveals with animated camera movements and scene transitions - all set up visually in Unity's Timeline without writing code.

## Use Cases

Build polished product presentations, create marketing showcases with dramatic reveals, design animated portfolios, or craft engaging e-commerce experiences with professional camera work.

## What This Sample Shows

Cinematic camera animations are created using Unity's Timeline system and play smoothly on the web. The sample demonstrates:

- **Timeline Animation** - Camera flyovers and product reveals animated in Unity
- **Signal Events** - Trigger actions at specific points in the timeline
- **SceneSwitcher** - Seamlessly transition between different products and scenes

Everything is set up visually - no code required!
